# MBA AlmaU Impact Forum — Telegram Bot

Telegram-бот для **MBA AlmaU Impact Forum** (30-летие программы MBA AlmaU).

Собирает инсайты участников во время докладов, валидирует их с помощью ИИ (Gemini) и формирует аналитические отчёты для организаторов (DeepSeek).

## Что умеет бот

### Для участников
- 📅 **Программа форума** — расписание по дням (4–5 июня)
- 🎤 **Спикеры** — биографии ключевых спикеров
- 📓 **Личный блокнот** — заметки по спикерам, общие заметки и инсайты с пуш-опросов
- 🤖 **ИИ-валидация** — Gemini отсекает спам и помогает участнику сформулировать мысль

### Для организаторов
- 📢 **Пуш-опросы** — запуск опроса через админ-панель, рассылка всем участникам
- 🔔 **Рассылка уведомлений** — произвольные сообщения с картинками
- ⚙️ **Админ-панель** — веб-интерфейс для управления опросами и просмотра ответов
- 📊 **Excel-экспорт** — все инсайты сессии в таблице
- 🧠 **ИИ-аналитика** — DeepSeek выделяет тренды и ключевые мысли

Подробная инструкция из 5 простых шагов (без программирования): **[instructions.md](instructions.md)**

## Технологии

- Node.js + Telegraf
- Google Gemini 1.5 Flash API (валидация инсайтов)
- DeepSeek API (ИИ-анализ трендов)
- Vercel Serverless + Vercel Blob Storage
- xlsx (Excel-экспорт)

## Структура файлов

```
api/
├── webhook.js      # Логика бота (команды, кнопки, обработка текста)
├── db.js           # Хранилище данных (Vercel Blob + локальный файл)
├── push-poll.js    # Рассылка опросов
├── analytics.js    # Excel и ИИ-аналитика
├── admin.js        # Admin API (сессии, инсайты, рассылка, Excel)
└── setup.js        # Автонастройка вебхука (открыть в браузере)

public/
├── index.html      # Лендинг
├── admin.html      # Админ-панель (веб-интерфейс)
└── favicon.svg     # Иконка
```
