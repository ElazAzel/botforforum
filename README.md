# MBA AlmaU Impact Forum — Telegram Bot (Meta-Harness)

Этот проект представляет собой готовое решение «под ключ» для сбора, валидации и семантического анализа инсайтов участников юбилейного форума **MBA AlmaU Impact Forum**. 

Проект построен по методологии **Meta-Harness** с использованием принципа **Zero-Cost** (нулевые затраты на инфраструктуру) и работает в среде Vercel Serverless с автоматическим выбором базы данных (локальный файл `db.json` для разработки или облачный `Vercel KV` для продакшена).

## ✨ Основной функционал

1. **Интерактивное меню участника**:
   * **Программа форума**: Разделена по дням (День 1 — 4 июня, День 2 — 5 июня) с возможностью быстрой навигации и кнопками возврата.
   * **Спикеры**: Список ключевых спикеров форума (Павел Лукша, Тарик Курейши, Ильдар Валиуллов и др.) с их биографиями.
   * **Мой блокнот**: Личный кабинет участника. Накапливает все принятые инсайты. Если длина превышает 4000 символов, бот генерирует `.txt` файл в памяти и отправляет его документом.
2. **Пуш-опросы (Meta-Harness Feedback Loop)**:
   * Администратор может инициировать пуш-опрос по конкретной сессии через эндпоинт `/api/push-poll`.
   * Бот активирует сессию и рассылает вопрос всем зарегистрированным пользователям, переводя их в режим ожидания ответа.
3. **Интеллектуальная валидация (Interviewer-Agent)**:
   * Каждый ответ пользователя отправляется в **Gemini 1.5 Flash**.
   * ИИ отсекает флуд, спам, пустые или односложные фразы (например, «нормально», «ок»).
   * Если инсайт прошел валидацию, он форматируется и дозаписывается в личный блокнот участника. Если не прошел — ИИ дает конструктивную обратную связь с просьбой дополнить мысль.
4. **Смысловая аналитика для организаторов**:
   * **Экспорт в Excel**: Эндпоинт `/api/analytics` с параметром `action=excel` собирает все инсайты сессии, генерирует Excel-файл `.xlsx` в оперативной памяти и пересылает его организатору.
   * **ИИ-анализ трендов**: Эндпоинт `/api/analytics` с параметром `action=trends` склеивает массив инсайтов и отправляет в **DeepSeek V3/V4**. ИИ выделяет ТОП-5 сквозных трендов, 3 уникальные/критические мысли и формирует 10 концептуальных слов для облака тегов.

## 🛠 Технологический стек

* **Runtime**: Node.js (версия >= 18.0.0)
* **Telegram Bot API**: библиотека `telegraf` (версия 4.x)
* **ИИ Модели**:
  * **Interviewer-Agent**: Google Gemini 1.5 Flash API (структурированный вывод через JSON Schema).
  * **Semantic-Analyst-Agent**: DeepSeek V3/V4 API (OpenAI-совместимый формат).
* **База данных**:
  * *Локально*: Автоматический текстовый файл `db.json` в корне проекта (удобно для отладки).
  * *Продакшн*: Бесшовное переключение на облачный `Vercel KV` (Upstash Redis) через быстрые прямые REST-запросы без лишних библиотек подключения.
* **Экспорт**: библиотека `xlsx` (генерация таблиц в памяти).

## 🎯 Быстрый старт

1. Склонируйте репозиторий.
2. Установите зависимости:
   ```bash
   npm install
   ```
3. Запустите локальный сервер:
   ```bash
   npm start
   ```
4. Настройте ngrok и установите вебхук для вашего бота.
