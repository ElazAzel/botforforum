# 🚀 Инструкция по запуску бота (4 простых шага)

Никаких терминалов, серверов, установок. Только браузер.

---

## Шаг 1: Подключите GitHub к Vercel

1. Откройте [vercel.com](https://vercel.com) и нажмите **Sign Up** (или **Log In**, если аккаунт есть).
2. Войдите через **GitHub** (кнопка «Continue with GitHub»).
3. На главной странице нажмите **Add New... → Project**.
4. Найдите репозиторий **botforforum** в списке и нажмите **Import**.
5. Нажмите **Deploy** — больше ничего менять не нужно.
6. Подождите 1-2 минуты. Vercel покажет зелёную галочку ✅ и ссылку вида `https://botforforum-xxxxx.vercel.app`.

---

## Шаг 2: Добавьте API-ключи

1. На странице вашего проекта в Vercel нажмите вкладку **Settings** (вверху).
2. В левом меню нажмите **Environment Variables**.
3. Добавьте следующие переменные (по одной — вставляете Name и Value, нажимаете **Add**):

| Name | Value |
|---|---|
| `TELEGRAM_BOT_TOKEN` | *(ваш токен бота от @BotFather)* |
| `GEMINI_API_KEY` | *(ваш ключ от Google AI Studio)* |
| `DEEPSEEK_API_KEY` | *(ваш ключ от DeepSeek)* |
| `DEEPSEEK_API_URL` | `https://api.deepseek.com/v1/chat/completions` |
| `ADMIN_PASSWORD` | *(придумайте пароль для админ-эндпоинтов)* |

> 💡 Все ваши ключи хранятся в файле `.env` на вашем компьютере. Скопируйте значения оттуда.

4. После добавления всех переменных перейдите во вкладку **Deployments**.
5. Найдите последний деплой, нажмите на **три точки** (⋯) справа → **Redeploy** → **Redeploy**.

---

## Шаг 3: Активируйте бота

1. Подождите, пока Vercel закончит повторный деплой (1-2 минуты).
2. Откройте в браузере адрес:
   ```
   https://ваш-проект.vercel.app/api/setup
   ```
   (Замените `ваш-проект` на реальное имя из Vercel — например, `botforforum.vercel.app` или `botforforum-xxxxx.vercel.app`).
3. Вы увидите зелёное сообщение **«✅ Вебхук успешно установлен!»**.

---

## Шаг 4: Проверьте бота

1. Откройте Telegram.
2. Найдите вашего бота (по имени, которое вы задали в @BotFather).
3. Отправьте команду `/start`.
4. Бот ответит приветствием и покажет меню с кнопками:
   - 📅 Программа (День 1 / День 2)
   - 🎤 Спикеры
   - 📓 Мой блокнот

**Бот готов к работе! 🎉**

---

## 📢 Как запустить опрос во время доклада

Когда спикер закончит выступление и вы хотите собрать инсайты, отправьте POST-запрос на:

**URL:** `https://ваш-проект.vercel.app/api/push-poll`

**Тело запроса (JSON):**
```json
{
  "session_id": "session_1",
  "title": "Форсайт-лекция Павла Лукши",
  "admin_password": "ваш_пароль_админа"
}
```

Бот разошлёт всем участникам вопрос и начнёт принимать их ответы.

## 📊 Как получить аналитику

**Excel-отчёт** (все инсайты сессии в таблице):
```
https://ваш-проект.vercel.app/api/analytics?action=excel&session_id=session_1&admin_tg_id=ВАШ_TELEGRAM_ID&admin_password=ваш_пароль_админа
```

**ИИ-анализ трендов** (ТОП-5 трендов, уникальные мысли, облако тегов):
```
https://ваш-проект.vercel.app/api/analytics?action=trends&session_id=session_1&admin_tg_id=ВАШ_TELEGRAM_ID&admin_password=ваш_пароль_админа
```

> 💡 **Как узнать свой Telegram ID?** Напишите боту @userinfobot в Telegram — он пришлёт ваш цифровой ID.
